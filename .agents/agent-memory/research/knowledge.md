## 1. SEED事業の根本目的

**ミッション**：海外にIPを輸出できる**プロデューサー・劇作家・演出家**を育成する

- 運営主体：JPASN（Japan Performing Arts Support Network）/ 緊急事態舞台芸術ネットワーク
- 委託元：文化芸術振興基金（芸文振）
- 事務局拠点：Port（ポート）
- 運営期間：第Ⅰ期（2025年7月〜2028年3月）

**業界課題認識**：日本発の舞台芸術作品・人材が海外で持続的に評価され、経済的利益を生む環境が整っていない。

## 7. SEED事業の全体像（リサーチエージェントとしての文脈理解）

### SEEDとSOILの関係

- **SOIL→SEED**：SOILが海外ピッチ等で得た市場ニーズをSEEDのカリキュラムに反映
- **SEED→SOIL**：SEEDで育成した人材・作品をSOILの海外派遣プログラムへ送り込む

### 国際展開戦略フレームワーク（SOIL根幹）

**→ 詳細ドキュメント：`0_docs/soil_strategy/international_strategy_framework.md`**

このフレームワークはSEED・SOIL両事業の根幹となる市場認識・構造認識であり、カリキュラム設計の大前提。2026年6月1日にClaude（Anthropic）との議論を通じて生成され、正式フォーマットPDF（39ページ）として整備済み。

**フレームワークの核心概念（要約）：**

- **4列の市場構造**：国際IP → 商業招聘 → 現代演劇 → 文化資産
- **転生装置**：文化機関と商業劇場の中間に位置し、作品を商業転生させる機関（NT、Barbican、Public Theater等）
- **3つの入り口**：A. 資本で場所を買う / B. 転生装置と共同制作 / C. フェスティバルから這い上がる
- **4つの戦略シナリオ**：トトロ型・ストンプ型・木下歌舞伎型・チェルフィッチュ型
- **最大の空白**：国際IP領域に日本がいない → SEEDが育成すべき人材の方向性を示す

**カリキュラムとの接続**：各シナリオが「プロデューサー・演出家・劇作家それぞれに何が必要か」を具体的に規定している。人材要件定義の際はこのフレームワークを必ず参照すること。
