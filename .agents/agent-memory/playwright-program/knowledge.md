## 1. プログラムのコンセプト

**対象**：18〜25歳の若手劇作家（大学生・大学院生・セミプロ）

**目的**：
> 日本の劇作における共通知識の不足を解消し、海外展開するためのIPとしての戯曲を書くことを目的とする。

**目指す作品像**：
- 言語の壁を乗り越えられる（「日本特有」の要素が作品の根本にならない）
- 上演後「Japaneseだから」ではなく「この劇だから」観て良かったと思われる
- アダプテーションの場合は「今上演することに意義がある」

**反面教師ケース**：KAJIMOTOとScottish Operaの共同制作『The Great Wave』  
→「日本への関心の高さ」は証明されたが、内容への評価は3 stars以下。日本的な表層だけでなく物語の完成度が問われた。

### 目指す劇作家像

| 要件 | 内容 |
|---|---|
| 基本知識 | 劇作の構造・理論の基本知識（海外との共通言語として） |
| テーマ意識 | 今の世の中で上演するのに相応しいテーマへの知識と情熱 |
| 柔軟性 | 他者・海外との働き方・創作プロセスの違いに対応できる |
| IP理解 | 海外に通用するIPとは何かの理解（海外観客層の需要を知る） |
| 言語化力 | 自分のアイデア・意志を言語化できる。作品への自信 |
| プロセス理解 | 上演までのプロセスを知っている |

## 2. 業界課題（プログラムが必要な理由）

1. **劇作に関する共通知識がない**：「すごい劇」の何がすごいか言語化できない。戯曲翻訳も独学で各自バラバラに行っている（言語・文化・スキル・コミュニケーションすべてで）
2. **プロとの出会いの場がない**：現役劇作家と話すだけで将来への解像度が一気に上がる
3. **安心して挑戦できる場がない**：社会のレールから離れることへの不安 → ちゃんとしたプログラムがあれば安心して劇作に挑める
4. **翻訳の重要性が認識されていない**：費用の問題で古い訳が使い回され、戯曲翻訳は賞味期限が最も短い（口語のため）

### 4-B. オンライン実施の検討（企画書docxより）

| 観点 | 内容 |
|---|---|
| メリット | 運営負担の軽減／遠方の人が気楽に参加できる／浮いた交通費を別用途に使える |
| デメリット | 集中力を保ちにくい／講義前後の自然な会話・個別質問の機会が失われる／「生身の人間」が根源の芸術を映像で教える違和感／自作をピッチ・共有する対面経験が自信につながる |
| 折衷案 | 完全オンラインにせず、**初回と最終回だけでも対面**にするだけで大きな違い |

**オフラインでないと育成が難しいもの**
- スキル：自作を語る・言語化する力／他者のフィードバックを受け取る力（Writing exercise で対面環境が重要）
- 経験：演劇の現場を体験してから書く／書いた作品が俳優に声に出される経験（Introductory Courseで5分程度のショートプレイを役者に読んでもらう）／翻案作品の制作・上演／自作の国際リーディング・英語上演／ノンバーバル・devising体験

## 8. Phase 2：Traverse Theatre リーディング（2027年9月）

### 概要

- **場所**：Traverse Theatre（エディンバラ、スコットランド）
  - Artistic Director：Gareth Nicholls
- **形式**：Rehearsed Reading / Work in Progress（Full Productionも視野）
- **時期**：2027年9月（7月は大学学期中のため渡英困難）
- **参加**：選考2通過の劇作家 最大5名が渡英
- **予算目安**：約475万円（育成検証企画PPTX）

### Traverseとの交渉結果（2026年5月14日MTG）

| 項目 | 内容 |
|---|---|
| 戯曲の長さ | 1時間程度が適切（2時間は経験の少ない劇作家にはハードルが高い） |
| 登場人物制限 | 事前に人数制限を設ける方が書きやすく上演可能性も上がる（例：役者3名まで） |
| Rehearsed Reading予算 | 役者レート：1日£200。人数次第で調整可 |
| Full Production予算 | £40,000〜£90,000（1人芝居で£40,000・3週間稽古+1週間テクが目安） |
| Full Productionスケジュール | 半年〜1年前に戯曲を読んでもらい上演判断・プログラムに組み込み |
| 国際コラボへの姿勢 | 積極的（British Councilサポート実績あり、Quebec劇団との戯曲exchange等） |
| 一点懸念 | 第2選考からProductionまでのスケジュールにもう少し余裕が必要 |

### 渡英中の内容

- Traverse Theatreでリーディング上演
- 役者とのコミュニケーション
- 観客のリアクション確認・フィードバック取得
- 現地での観劇
- 現地演出家・スタッフ・演劇専攻学生との交流会
