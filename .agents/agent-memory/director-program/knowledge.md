## 2. コンセプト・目指す姿

### 業界の入口問題

> 「業界として、入口が一定あるのはプロデューサー。劇作家、演出家の入口はないが、どうする？」（定例 12/4）

→ SEEDが演出家・劇作家の「入口」を作る役割を担う。

### 3職種の接続意義

> 「プロデューサーが劇作家、演出家と出会う意義はある」

→ サマースクールは3職種が交差して初めて意味を持つ設計。

### 目指すべき人材のベンチマーク

**「海外の舞台演出専攻の大学院に合格できるような人材」** をベンチマークとして設定する。

これは最終的な進学を目的とするのではなく、以下の観点からの基準値として採用する：

- 海外大学院の演出専攻が求める**ポートフォリオ・演出ステートメント・批評的思考**のレベルを到達目標の参照軸とする
- 欧米の審査基準（Yale School of Drama、RADA、East 15 等）を意識したカリキュラム設計・選考設計を行う
- 受講生が「自分の演出的視点を英語で言語化できる」水準を目安とする

## 3. フェーズ1：講座シリーズ

### 演出家コースの位置づけ

- 社会人も参加可能なオープン講座形式
- 演出家・プロデューサーは**選考テストのために参加推奨**（必須ではなく任意）
- 劇作家は一部講座参加が必須（演出家は任意扱いで差異あり）

### 講座内容の方向性（リサーチ知見より）

リサーチャー（小栗・田中・上田）の共通知見として川名さんが報告（3/23定例）：

| 知見 | 内容 |
|---|---|
| **メンタリングが最重要** | 3名のリサーチに共通する核心。「メンター誰がいいのか」（候補は未定） |
| **他の作品を観ることの重要性** | 映像でも可。観劇・鑑賞習慣の形成 |
| **演劇メソッドの基礎** | 「演劇メソッド（4つ）、身体メソッドの基礎について教えるにはいい機会かも」（検討中） |
| **自分の哲学に関する訓練** | 「メソッドに関する内容入れ込む？（自分の哲学に関する訓練）」（検討中） |

### 対応する演出家ToBeモデルの最優先要件

（`competency`エージェントのナレッジより）

| 要件 | 講座での扱い方（案） |
|---|---|
| K-2 演技メソッド・共通言語 | 演劇メソッド4種・ラバン等の概論 |
| K-3 演劇史・世界の演出家への理解 | 世界の演出家・演劇運動の講義 |
| K-4 日本の文化・伝統芸能 | 日本演劇の歴史と現在（一般教養と連携） |
| S-3 ビジョンの言語化・共有力 | 演出プランの書き方・発表ワーク |

## 4. フェーズ2：Paidインターン

### 基本方針（4/20定例より）

> 「学生に提供したいものとしては『現場に居る』こと」  
> 「みんなの話が聞こえる場所で」「様々な打ち合わせにJOIN」

- **理想**：海外演出家の現場
- **規模感**：毛色の違う3件程度のプロダクションに参加
- **方針**：「1人が10回行く」ではなく、複数現場・複数人に広く経験させる
- **タイミング**：特定時期に縛らない（「割といつでも良いかも」）
- **1日からでも可**：「1日でもOK！理想はもっと」

### インターン受け入れ候補（検討中）

- **埼玉芸術劇場**（候補として浮上）
- **世田谷パブリックシアター**（候補として浮上）
- **ワークショップのアシスタント**（形態の一つとして）

**現状**：各所への種まきフェーズ。2026年7月27日のJPASN会員総会が受け入れ先確保の追い込み目標。

## 6. 選考設計（未確定事項多数）

### 確定事項

- 選考あり（書類 + 面接）
- 「ハードルが高くて良い予感」（高い基準で選ぶ方針）
- 選考通過後、キャンプ開催までに事前準備プログラムを予定

### 未確定事項

> 演出コースの試験内容は**白紙**のまま（「演出プランとは」と書かれたまま未決定）

→ 演出プランの提出を求める方向性はあるが、具体的な要件・評価基準が未設計。

## 10. このエージェントの優先タスク

1. **メンタープログラムの設計**（誰がメンターになるか・マッチング方法・セッション設計）
2. **選考試験の設計**（演出プランの提出要件・評価基準の言語化 → `selection`と連携）
3. **講座シリーズの内容設計**（演劇メソッド4種の選定・観劇プログラム・ビジョン言語化ワーク）
4. **インターン先のリスト化と交渉支援**（埼玉芸術劇場・世田谷PT等への依頼内容の設計）
5. **劇作家プログラムとの合同セッション設計**（テクスト×演出の対話の場）
6. **海外演出家インターンの実現可能性整理**（最も理想的だが最も難易度が高い）
