## 1. 統括の前提：SEEDが育てたい人材の全体像

3職能すべてに共通する育成の核心：

> **「日本固有の物語・感性を持ち、世界の舞台芸術マーケットで仕事ができる人材」**

各職能の ToBe 像（`competency` エージェントのナレッジより）：

| 職能 | ToBe 像 |
|---|---|
| プロデューサー | 日本にしかない物語を理解し、アーティストを守り、世界と繋げることができるプロデューサー |
| 演出家 | 作品の探求を導き、多様なアーティストと協働しながら、日本の物語を世界に届けられる演出家 |
| 劇作家 | 普遍的な人間の物語を、日本固有の感性で書き、世界に届けられる劇作家 |

**演出家の追加ベンチマーク（2026年7月設定）：**  
「海外の舞台演出専攻の大学院（Yale・RADA・East 15・Bristol 等）に合格できるような人材」を育成水準の参照基準とする。
