## 1. プロデューサープログラムの全体コンセプト

**テーマ：**「あなたが手がけた作品を観たい」

**目指す人材像：**
> 「エコシステムを育てる『利他的』な視点を持ち、自分（自団体）の成功だけでなく、日本の舞台芸術界全体のプレゼンスを高める自律したプロデューサー像」

### 目標コンピテンシー（4軸）

| 軸 | 具体的要件 |
|---|---|
| **知識・教養** | 国際的な法務・契約・税務知識／知的財産の知識／各国の助成・投資メカニズム／言語の壁を越えるテクノロジー理解／社会情勢・各国の文化と日本の独自性 |
| **マインド** | 商習慣や文化の違いを受け入れる柔軟性／対等なパートナーシップを築けるマインド／獲得したノウハウを若手育成・業界全体へ還元する公共的な志 |
| **創造力** | 現地のニーズを汲み取り舞台作品の社会的価値を設計する力／海外展開可能なコンテンツを育む力／国内既存作品のローカライズができる力 |
| **共創力** | クリエイターとコラボレートし作品の価値を最大化できる力／戦略的なネットワーキング／的確にプレゼンテーションできる力／ステークホルダーと対話し利益を最大化できる力 |

## 2. プログラム全体構造（3フェーズ）

```
【フェーズ1】基礎（9〜10月）
  座学・ワークショップ + マインドセット形成

【フェーズ2】実践（1〜3月）
  Paidインターン + メンターシップによる現場経験

【フェーズ3】応用
  劇作家・演出家コースと共同でクリエイション + ピッチプレゼンテーション
```

## 3. フェーズ1：座学オープン講座（9〜10月）

### 実施概要

- **形式**：オンライン×対面ハイブリッド
- **定員**：20名（旅費補助制度を検討中）
- **位置づけ**：実践編の選考筆記テストの出題範囲 + 面接評価軸

### カリキュラム（6テーマ）

| No. | テーマ |
|---|---|
| 01 | 舞台芸術プロデューサーの役割と歴史 |
| 02 | 著作権・契約・IP基礎 |
| 03 | 海外展開の法務・税務 |
| 04 | アクセシビリティと多様性 |
| 05 | 資金調達（助成金・スポンサー・クラウドファンディング） |
| 06 | マーケティングと観客開発 |

### 特別セッション（各90分・質疑含む）

- **A. 先輩プロデューサーから学ぶ**
- **B. 海外プロデューサーの日本への期待値を知る**（4か国程度）

## 4. フェーズ2：実践編（1〜3月）

### 4-1. インターン（海外演出家プレプロダクション型）

- **形式**：海外演出家のプレプロダクション制作型インターン（3〜5本）
- **参加人数**：1プロダクションあたり2名（プレプロ担当1名 + WS担当1名）× 3〜5現場
- **期間**：5日間程度のプレプロダクション

**役割分担：**

| 主体 | 役割 |
|---|---|
| 協力企業 | 海外演出家の紹介、スタッフ・キャストとの接続 |
| インターン参加者 | プレプロ制作進行、キャスト調整、打ち合わせ同席 |
| SEED | 演出家来日費用負担、参加者フォローアップ |

**実施内容（プレプロダクション内）：**
- プロの俳優による本読み
- スタッフとのプロダクションミーティング
- ワークショップ開催

**内部設計の意図：**
- 参加者2〜3名に対し1名のSEED選出伴走者（→将来の育成担い手候補）
- 企業側に明確なメリットを持たせ、企業にも責任とイノベーションを
- 参加者が考えたアイデアを演出家と話せる機会も設けたい

### 4-2. メンターセッション（企画の種を育てる・1〜2月）

**メンター体制：**

| メンター | 所属 | 備考 |
|---|---|---|
| 小嶋麻倫子 | 東宝 | SEEDリサーチ派遣者 |
| 井川荃芬 | ホリプロ | SEEDリサーチ派遣者（NY渡航済） |
| 時田曜子 | ゴーチ・ブラザーズ | SEEDリサーチ派遣者（ロンドン予定） |

- 1メンターあたり受講生3名程度

**プログラム内容（ステップ）：**

1. 受講生が「海外展開を実現したいアイデア」を企画書化
2. 選考（座学テスト + 面談）
3. 選考後：企画書ブラッシュアップ → キャスト・スタッフ具体案 → 予算書作成 → テクニカルライダー作成

**コンセプト：**「プロデューサーがIPの種を生み出し、サマースクール（キャンプ）に向けての準備を」

## 5. フェーズ3：応用（クリエイション + ピッチ）

- 劇作家・演出家コースのメンバーと共同でクリエイション
- 立ち上がりのある作品のピッチプレゼンテーション
- → サマースクールの集大成として機能させる構造

---

## 6. 選考設計

- **選考方式**：筆記テスト + 面談の2段階
- **筆記テスト範囲**：座学オープン講座（フェーズ1）の6テーマから出題
- **面接評価軸**：座学の学習観点 + コンピテンシー4軸（知識・マインド・創造力・共創力）
- **サマースクール全体の選考体制**：
  - 選考①：一般教養テストを含む一次選考
  - 選考②：職能別の二次選考

## 7. プロデューサープログラムに関する議論・課題（定例より）

### アドバイザリーボードからの宿題（4/20定例）

- 人材要件の「下流レイヤー」の具体化が求められた
  - 例：「企画書構成力」「数字を作る力」等の実務スキル項目
- **横堀さんの宿題（3点）**：
  1. 国内プロデューサーのAs-Is人材要件定義（国内Pへのヒアリング含む）
  2. To-be人材要件のブラッシュアップ
  3. ①②の差分をSEEDプログラムで埋めるという説明の構造化

### Paidインターンの位置づけ議論（4/27定例）

未解決事項として継続議論中：
- 初年度はリサーチャーの現場に受け入れてもらう案（野村さん）
- SEEDプレゼンツの公演制作の方が向いているかも（赤羽さん）
- 国外を目指す事業の中での国内派遣の矛盾（伊藤さん提起）
- 海外を目指す上での国内ファーストステップとしての意義（川名さん）

### 一般教養との接続（内部メモ）

- 座学オープン講座は「一般教養と重なる？」という内部検討あり
- → 一般教養講座との内容整合・棲み分けの設計が必要

## 9. このエージェントの優先タスク

1. **座学カリキュラム詳細設計**（6テーマ×各セッションの内容・目標・時間配分）
2. **講師ブリーフの作成**（各ゲスト候補への依頼内容の整理）
3. **選考テスト問題の設計**（座学6テーマからの出題設計 → `selection`エージェントと連携）
4. **Paidインターン設計の議論整理**（位置づけの論点を整理し、方針決定を支援）
5. **一般教養との棲み分け整理**（重複テーマの調整 → `general-education`エージェントと連携）
6. **人材要件の下流レイヤー詳細化**（「企画書構成力」「数字を作る力」等 → `competency`エージェントと連携）
