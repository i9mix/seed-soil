## 1. プロデューサープログラムの全体コンセプト

**テーマ：**「あなたと興行がしたい／あなたが手がけた作品を観たい」

**目指す人材像：**
> 「エコシステムを育てる『利他的』な視点を持ち、自分（自団体）の成功だけでなく、日本の舞台芸術界全体のプレゼンスを高める自律したプロデューサー像」

### 目標コンピテンシー（4軸）

| 軸 | 具体的要件 |
|---|---|
| **知識・教養** | 国際的な法務・契約・税務知識／知的財産の知識／各国の助成・投資メカニズム／言語の壁を越えるテクノロジー理解／社会情勢・各国の文化と日本の独自性 |
| **マインド** | 商習慣や文化の違いを受け入れる柔軟性／対等なパートナーシップを築けるマインド／獲得したノウハウを若手育成・業界全体へ還元する公共的な志 |
| **創造力** | 現地のニーズを汲み取り舞台作品の社会的価値を設計する力／海外展開可能なコンテンツを育む力／国内既存作品のローカライズができる力 |
| **共創力** | クリエイターとコラボレートし作品の価値を最大化できる力／戦略的なネットワーキング／的確にプレゼンテーションできる力／ステークホルダーと対話し利益を最大化できる力 |

## 2. プログラム全体構造（3フェーズ）

```
【フェーズ1】基礎（9〜10月）
  座学・ワークショップ + マインドセット形成

【フェーズ2】実践（1〜3月）
  Paidインターン + メンターシップによる現場経験

【フェーズ3】応用
  劇作家・演出家コースと共同でクリエイション + ピッチプレゼンテーション
```

## 2-B. V2講座案の最新アップデート（2026年7月）

### 目的（明確化）

> **日本発のコンテンツを海外展開可能な人材の育成（外貨を稼ぐ）。**

### 必要な能力（仮説・3点）

| 能力 | 内容 |
|---|---|
| 「これを世界に届けたい」という意思と審美眼 | 何がなぜ面白いのかを自分の言葉で語れる。市場に合わせて作品を変形させるのではなく、作品の核を守りながら届け方を変えられる判断軸 |
| 「交渉できる人間力」 | 契約書が読めることより、相手と信頼関係を作れること。文化的背景が違う相手と本音で話せる。語学力とは別に「自分が何者か」を持っている |
| 「お金の構造を設計できる能力」 | 助成金とビジネス成立は別。誰がどこでどうお金を出し、どう回収するかを複数シナリオで描ける（**日本のプロデューサーに最も欠ける部分**） |

### 講座設計におけるジレンマ

- 日本での成功例が少なく、国内で実地経験を積む場所が限られている
- 育成された人材が国内で活躍できる場が限られている

### 3フェーズの位置づけ（ラベル精緻化）

- **Phase 1 基礎・座学**（知識の獲得）
- **Phase 2 実践・ゼミ**（企画の立ち上げとマインドセット）
- **Phase 3 応用**（クリエイションとプレゼンテーション）
- スケジュール射程：2026年〜2028年。座学（一般教養）→ゼミ→プレ合宿（サマースクール）→IP展開・接続、**SEED 2期**へ接続。

### オープン講座 vs 集中講座（V2で追加された整理）

| 区分 | 対象 | 位置づけ |
|---|---|---|
| オープン講座 | 学生中心・興味がある人は誰でも | 企画の宣伝／人材との出会い |
| 集中講座 | 大学生 | 未来に向けての研究 |

**オープン講座の構成案**
- ① 未来のプロデューサー像とは？（座談会・登壇：小嶋／井川／時田＋聞き手）※舞台芸術おしごとナビに接続
- ② 特別講義A（IP×契約）：登壇候補 プペル／トトロ／ライオンキング 関係者
- ③ 特別講義B（ノンバーバル×フリンジ）：登壇候補 松田誠
- ④ 特別講義C（海外招聘×フェスティバル）：登壇候補 検討中
- SEED Lab群（マーケティング等）：ラウンドテーブル／グループワーク。進行＝SEED事務局（髙田／赤羽）

### リサーチャーゼミ（Phase 2 の中核・1〜3月）★V2の目玉

> ゼミリーダー（リサーチャー）を中心に「大学の研究室」的に運営。所属会社に関わらず、リサーチャーが研究したいテーマを深掘りする場。

**ゼミリーダー（4名）**

| リーダー | 所属 | 派遣先 |
|---|---|---|
| 井川 荃芬 | ホリプロ | ニューヨーク |
| 小嶋 麻倫子 | 東宝 | ニューヨーク |
| 時田 曜子 | ゴーチ・ブラザーズ | ロンドン |
| 田窪 桜子 | unrato | 韓国 |

**実施概要**
- 1ゼミあたり5名（×4ゼミで最大20名）
- 1回1時間半 × 6コマ（3〜5日間程度／約9時間）。うち1コマは JPASN 内公開
- プログラム内容はリサーチャーと相談の上決定。ゼミ講師は相互のゼミに参加可（事前合同MTG）。全ゼミ横断の観劇プログラムも検討（リサーチャー演目推薦）
- **スケジュール**：09月下旬 募集開始／11月上旬 締切／11月下旬 面談／12月中旬 参加者決定
- **募集**：海外展開に関心があり舞台芸術プロデューサーとして活躍したい大学生・院生・専門学校生。基本全回参加・終了後レポート提出

**ゼミが学生に与える価値（マップ）**
- 知識：企画の立ち上げ／テーマに基づく知見／クリティカルシンキングの訓練
- 視野：業界横断ネットワーク／海外展開を目指すリーダーとの接点／ロールモデルの獲得
- マインド：ゼミリーダーと同じ視座／自分の言葉で語る力／個人として評価される体験
- 接続：組織に依存しない自律したプロデューサー像の獲得／専門家との接点

**論点・留意**：ゼミはゼミリーダー依存（スペシャリスト養成）になりがち／リーダーが多忙／「エキスパートの集合体」の方が機能する可能性。「機会を待つだけでなく機会を作れる人間」「演劇を教材に国際競争力をつける」方向。

### フェーズ2の代替案（インターンに代わる案）

- **観劇プログラム**：事務局指定の演目を観劇 → 30分でレポートシート → プロデューサーとのセッション（60分）→ 同コンテンツで企画書（予算配分・ターゲット・チケット料金・キャッチコピー・次なる展開案）
- **TOEIC的なテスト**の施行と教材作り（座学を通じた教材知見収集・効果測定と検証）
- **プロデューサー版しくじり先生**：海外展開における失敗例を徹底的に聞く

## 4. フェーズ2：実践編（1〜3月）

### 4-1. インターン（海外演出家プレプロダクション型）

- **形式**：海外演出家のプレプロダクション制作型インターン（3〜5本）
- **参加人数**：1プロダクションあたり2名（プレプロ担当1名 + WS担当1名）× 3〜5現場
- **期間**：5日間程度のプレプロダクション

**役割分担：**

| 主体 | 役割 |
|---|---|
| 協力企業 | 海外演出家の紹介、スタッフ・キャストとの接続 |
| インターン参加者 | プレプロ制作進行、キャスト調整、打ち合わせ同席 |
| SEED | 演出家来日費用負担、参加者フォローアップ |

**実施内容（プレプロダクション内）：**
- プロの俳優による本読み
- スタッフとのプロダクションミーティング
- ワークショップ開催

**内部設計の意図：**
- 参加者2〜3名に対し1名のSEED選出伴走者（→将来の育成担い手候補）
- 企業側に明確なメリットを持たせ、企業にも責任とイノベーションを
- 参加者が考えたアイデアを演出家と話せる機会も設けたい

### 4-2. メンターセッション（企画の種を育てる・1〜2月）

**メンター体制：**

| メンター | 所属 | 備考 |
|---|---|---|
| 小嶋麻倫子 | 東宝 | SEEDリサーチ派遣者 |
| 井川荃芬 | ホリプロ | SEEDリサーチ派遣者（NY渡航済） |
| 時田曜子 | ゴーチ・ブラザーズ | SEEDリサーチ派遣者（ロンドン予定） |

- 1メンターあたり受講生3名程度

**プログラム内容（ステップ）：**

1. 受講生が「海外展開を実現したいアイデア」を企画書化
2. 選考（座学テスト + 面談）
3. 選考後：企画書ブラッシュアップ → キャスト・スタッフ具体案 → 予算書作成 → テクニカルライダー作成

**コンセプト：**「プロデューサーがIPの種を生み出し、サマースクール（キャンプ）に向けての準備を」

## 5. フェーズ3：応用（クリエイション + ピッチ）

## 6. 選考設計

- **選考方式**：筆記テスト + 面談の2段階
- **筆記テスト範囲**：座学オープン講座（フェーズ1）の6テーマから出題
- **面接評価軸**：座学の学習観点 + コンピテンシー4軸（知識・マインド・創造力・共創力）
- **サマースクール全体の選考体制**：
  - 選考①：一般教養テストを含む一次選考
  - 選考②：職能別の二次選考

## 7. プロデューサープログラムに関する議論・課題（定例より）

### アドバイザリーボードからの宿題（4/20定例）

- 人材要件の「下流レイヤー」の具体化が求められた
  - 例：「企画書構成力」「数字を作る力」等の実務スキル項目
- **横堀さんの宿題（3点）**：
  1. 国内プロデューサーのAs-Is人材要件定義（国内Pへのヒアリング含む）
  2. To-be人材要件のブラッシュアップ
  3. ①②の差分をSEEDプログラムで埋めるという説明の構造化

### Paidインターンの位置づけ議論（4/27定例）

未解決事項として継続議論中：
- 初年度はリサーチャーの現場に受け入れてもらう案（野村さん）
- SEEDプレゼンツの公演制作の方が向いているかも（赤羽さん）
- 国外を目指す事業の中での国内派遣の矛盾（伊藤さん提起）
- 海外を目指す上での国内ファーストステップとしての意義（川名さん）

### 一般教養との接続（内部メモ）

- 座学オープン講座は「一般教養と重なる？」という内部検討あり
- → 一般教養講座との内容整合・棲み分けの設計が必要

## 9. このエージェントの優先タスク

1. **座学カリキュラム詳細設計**（6テーマ×各セッションの内容・目標・時間配分）
2. **講師ブリーフの作成**（各ゲスト候補への依頼内容の整理）
3. **選考テスト問題の設計**（座学6テーマからの出題設計 → `selection`エージェントと連携）
4. **Paidインターン設計の議論整理**（位置づけの論点を整理し、方針決定を支援）
5. **一般教養との棲み分け整理**（重複テーマの調整 → `general-education`エージェントと連携）
6. **人材要件の下流レイヤー詳細化**（「企画書構成力」「数字を作る力」等 → `competency`エージェントと連携）
