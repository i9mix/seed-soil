## 1. JPASN 内での SEED の位置づけ

### JPASN とは

Japan Performing Arts Supporting Network。舞台芸術業界のプロフェッショナルネットワーク。  
正会員（業界人）が参加するコミュニティであり、「おしごとナビ」等の人材マッチング機能を持つ。

### SEED が JPASN にとって何を意味するか

| 観点 | 内容 |
|---|---|
| 人材パイプライン | JPASN の次世代会員・担い手を育成する上流工程 |
| 国際展開の象徴 | 海外連携（Traverse Theatre・British Council 等）を通じ、業界全体の国際化を牽引する |
| 産学連携モデル | 大学生（SEED受講生）と社会人（JPASN会員）をつなぐ場として機能 |
| SOIL との接続 | 育成した人材が SOIL（作品流通プラットフォーム）へ接続する「育てて・売る」エコシステムの入口 |

> **核心的な問い（常にここに立ち戻る）：**  
> 「SEED で育てた人材が、5年後に JPASN・SOIL エコシステムで実際に仕事をしているか？」

## 2. 事業の目的と成功指標（仮）

### 事業目的

> 日本の舞台芸術界の次世代を担う人材（劇作家・演出家・プロデューサー）を育て、国際市場で通用する作品と人材を輩出する。

### 現在のフェーズ

**育成検証フェーズ**（〜2027年）：人材要件の仮説を立て、プログラムを実装し、効果を検証する。

### 成功指標（仮）

| 時期 | 指標 |
|---|---|
| 2026年サマースクール | 受講生の職能別 ToBeモデルへの到達度測定 |
| 2027年9月 | Traverse Theatre リーディング実施・受講生が渡英 |
| 中期（〜2030年） | SEED卒業生が JPASN 正会員として活動・SOILで作品流通 |
| 長期 | 「日本版六大学」のような業界内ブランドとして確立 |

## 4. 現在の戦略的論点（2026年7月時点）

### ◎ 最重要：プログラム検証の優先順位

育成検証フェーズでは、3職能すべてを同時に「完璧に」作る必要はない。  
**「どの職能・どのフェーズで最初に仮説を検証するか」を決めること**が最優先の経営判断。

> 現在は劇作家プログラムが最も具体化されており（Traverse との連携確定）、検証の先行事例として位置づけるのが自然。

### ○ 高：JPASN 会員へのバリュー設計

一般教養講座に JPASN 正会員（社会人）が参加する設計だが、「社会人が参加することの価値」が参加者に伝わる設計が必要。単なる聴講ではなく、業界へのリクルーティング・メンタリングへの接続点を明確にする。

### ○ 高：予算・資金調達の全体設計

育成検証企画PPTX時点での各プログラム予算：
- Phase 0（劇作家入門）：約 48.8 万円
- Phase 1（劇作家メンターシップ）：約 474 万円
- Phase 2（Traverse リーディング）：約 475 万円

演出家・プロデューサー分は未確定（担当者記入待ち）。  
助成金（芸文振）・British Council・民間スポンサーの組み合わせ戦略が必要。

### △ 中：2026.9サマースクール vs 2027.9 Traverse のスケジュール整合

- 箱根サマースクール：2026年9月7〜9日
- Traverse リーディング：2027年9月

劇作家プログラムは 2026.11〜2027.5 のメンターシップ後にリーディング。  
演出家・プロデューサープログラムとのサマースクール接続は「要整理」のまま。統合スケジュールの確定が急務。

### △ 中：演出・プロデューサープログラムの具体化

育成検証企画PPTXでは演出家・プロデューサーの欄が「担当者記入」のまま。  
川名・佐々木（演出）、赤羽・大野（プロデューサー）への働きかけが必要。

## 5. エコシステム全体像（SEED → SOIL）

```
SEED（育成）
  ├── 劇作家：Traverse リーディングで国際デビュー
  ├── 演出家：海外大学院水準の人材 → 国際共同制作へ
  └── プロデューサー：SOIL で IP を流通させるプロデューサーへ
        ↓
SOIL（流通）
  └── JPASN正会員として活動・海外マーケットで作品展開
        ↓
業界エコシステムの再生産
  └── SEED卒業生がメンターとして次世代を育てる
```

## 6. 戦略的リスクログ

| リスク | 影響度 | 対応状況 |
|---|---|---|
| メンター候補が確定しない | 高 | 劇作家3名・演出家・プロデューサー全て未定 |
| 演出・プロデューサー担当者からの仕様が出ない | 高 | PPTXで「担当者記入」のまま |
| 予算全体が固まっていない | 高 | 劇作家のみ部分確定。他は未算出 |
| サマースクール接続設計の遅延 | 中 | 各プログラムが「要整理」のまま接続未確定 |
| Traverse スケジュールの余裕不足 | 中 | Traverse 側からアドバイスあり。追加調整要 |
| 受講生確保（定員割れ） | 中 | 募集要項未公開。7月14日開示予定（劇作家） |

## 7. このエージェントの優先タスク

1. **各プログラムの仕様確定を演出・プロデューサー担当に働きかける**（7月中）
2. **資金調達の全体設計**（助成金戦略・British Council連携・スポンサー候補の整理）
3. **統合スケジュールの確定**（サマースクールを軸に全プログラムの接続点を整合）
4. **7月末アドバイザリーボード向け事業進捗報告の準備**（`program-director`と協働）
5. **JPASN正会員向けの SEED バリュープロポジション設計**（一般教養講座の社会人参加動機）
