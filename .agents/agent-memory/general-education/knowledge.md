## 1. 一般教養講座が生まれた背景

### 設立の直接の契機

> アドバイザリーボード定例で、演出・劇作コースに**知財のカリキュラムがなかった**ことから、福井健策弁護士（アドバイザリーボードメンバー）より「アーティストの知財の知識の必要性」を問われた。  
> **「プロデューサーだけが知っていてもクリエイションの際に話がスムーズに進まない。演劇に関わる人は広く必要な知識」** — SEED事務局

### 既存の大学教育の課題

> 「大学での一般教養は選択科目が多く共通言語になっていない」 — 芸文振（2026/3/18 MTG）

→ 3職種が協働するための**共通言語**を外部サマースクールで作る、というコンセプト。

## 2. 一般教養講座の位置づけ

- **サマースクール全体の共通土台**：職能別プログラムの前後に配置
- **産学連携の接点**：JPASN正会員（社会人）も参加できる設計（学生と社会人がつながる場）
- **選考との接続**：一般教養講座の内容をもとにした筆記テストが**選考①**に組み込まれる

```
一般教養講座
  ├── 3職種全員が受講（共通言語の形成）
  ├── JPASN正会員（社会人）も参加可能
  └── 内容が選考1の筆記テスト範囲に含まれる
```

## 3. 扱うべき内容（確定・検討含む）

### 確定テーマ（アドバイザリーボード・芸文振から示されたもの）

| テーマ | 根拠 |
|---|---|
| **知的財産・著作権** | 福井健策弁護士からの問題提起。演出家・劇作家も必須 |
| **海外展開の法務・契約基礎** | 3職種共通で必要。プロデューサー座学との重複に注意 |
| **ハラスメント対策・労働倫理** | JPASNのハラスメント対策PTとの連携検討あり |
| **英語コミュニケーション基礎** | 海外展開を目指す全職種に必要 |

### 検討テーマ（初年度振り返りMTG・定例より）

| テーマ | 根拠 |
|---|---|
| 舞台芸術の歴史・批評リテラシー | 3職種共通の文脈理解として有効 |
| 社会における舞台芸術の意義 | 「今上演する意義」を語れる力の土台 |
| IP（知的財産）の概念と海外展開 | 「IPとして戯曲・作品を考える」視点 |
| 資金調達の基礎（助成金・スポンサー） | 劇作家・演出家も知るべき実務知識 |
| 国際的な演劇トレンド | DEI・アクセシビリティ・LGBTQ+等の文脈理解 |
| 日本の文化・伝統芸能の基礎 | 海外への説明力の土台。3職種共通の最優先要件 |

### プロデューサー座学との棲み分け（重要）

プロデューサー講座（座学）と内容が重複するリスクあり。内部メモにも「一般教養と重なる？」との記載。

**棲み分けの考え方（案）：**
- **一般教養**：3職種が「知っておくべき」概念レベルの共通知識（Why・What）
- **プロデューサー座学**：プロデューサーとして「実践できる」スキルレベル（How）

## 4. プログラム設計の方向性

### 形式

- **ハイブリッド**（オンライン + 対面）
- サマースクール前の事前プログラムとして位置づけるか、本体と組み合わせるかは検討中

### 社会人参加の意義（芸文振MTGより）

> 「産学連携として学生が社会人とつながる時間はなかなかない」 — 芸文振  

JPASN正会員（業界人）が同じ講座を受けることで：
- 学生が業界人と同じ文脈・言語を持てる
- 業界人側のアップデートにもなる
- サマースクール後の就職・キャリア接続（JPASNの「おしごとナビ」）への布石

### 参加者の出口イメージ（芸文振MTGより）

> 「サマースクールでできた団体が劇団化し、JPASNに加盟し、SOILで採用されれば、日本版『six』のようなストーリーが描けるかもしれない」 — SEED事務局

## 5. 3職種共通の人材要件との対応

`competency`エージェントが整理した3職種横断の最重要要件を、一般教養で扱うべき内容に変換：

| 共通要件 | 一般教養での扱い方 |
|---|---|
| 日本の文化・伝統芸能への理解 | 日本演劇史・文化的価値観のレクチャー |
| 演劇の構造・ドラマトゥルギー基礎 | 戯曲読解・批評リテラシーの入門 |
| 制作実務基礎（助成金・契約） | 知財・著作権・資金調達の概論 |
| 英語コミュニケーション | 英語ピッチ・自己表現の基礎ワーク |
| 国際トレンドの理解 | DEI・アクセシビリティ・世界の演劇潮流 |
| コラボレーション・協働姿勢 | 職種横断グループワーク・対話の場 |

## 6. SOIL・業界エコシステムとの接続

初年度振り返りMTGで整理された「SEED → SOIL」の連携構造は、一般教養の**学習の動機付け**にもなる：

- SEEDで「なぜ海外展開が必要か」の文脈を共通言語として習得
- SOILが現場で作品を売る際に使う「アカデミックな裏付け・日本演劇の解説力」を、一般教養で下地として作る
- → 一般教養は「業界エコシステム全体への理解」を育てる場でもある

## 7. 現状の課題・未解決事項

1. **プロデューサー座学との棲み分けの明確化**：テーマ・深度・受講対象の設計が必要
2. **一般教養単独の資料がない**：`SEED_サマースクール_一般教養.pptx`はバイナリ形式のため未読。内容確認が必要
3. **社会人（JPASN会員）向けのコミュニケーション設計**：学生と社会人が混在する場のファシリテーション方針
4. **テスト（選考1）との連動設計**：講座の何を、どう出題するかの設計が必要

## 8. このエージェントの優先タスク

1. **一般教養カリキュラムの全体設計**（テーマ選定・順序・各セッションの目標と方法）
2. **プロデューサー座学との棲み分け整理**（重複テーマの調整 → `producer-program`と連携）
3. **3職種共通要件の「教養レベル」への変換**（`competency`エージェントの成果を活用）
4. **選考1の筆記テスト観点の設計**（→ `selection`エージェントに渡す）
5. **社会人参加を活かす場づくりの設計**（学生×社会人の接続ポイントの設計）
6. **`SEED_サマースクール_一般教養.pptx`の内容確認**（Googleスライドに変換してもらえれば読める）
