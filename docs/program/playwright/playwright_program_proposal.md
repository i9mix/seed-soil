## 概要

**Playwright Support Project — From Japan to the International Stage**

若い劇作家が構造化されたプログラムを通じて知識を習得し、同世代・プロフェッショナルとアイデアを共有する機会を創出する。

---

## 課題認識

| 課題 | 詳細 |
|------|------|
| 共通知識の欠如 | 劇作・演劇に関する共有知識ベースが存在しない |
| 新進劇作家へのサポート不足 | 新人劇作家を体系的に支援する仕組みがない |
| 国際ステージへの意識の低さ | 国際的な舞台を視野に入れた創作意識が育っていない |

---

## プログラム目標（成果物の条件）

最終作品には以下の要件を満たすことを求める：

- **現代社会に関連するメッセージ**を持つ
- 国境を越えて共有できる**普遍的な価値観**を有する
- 日本的要素が物語・効果を強化する役割を担いつつ、**それだけが強みではない**作品

---

## プログラム全体設計

### 対象者

**18〜25歳**の若い劇作家

### フロー

```
導入講座（Introductory Class）
        ↓
選考①（プロット提出ベース）
        ↓
フェーズ1：メンターシップ（Mentorship）
        ↓
選考②（メンター期中に執筆したドラフトベース）
        ↓
フェーズ2：プロダクション（Production）
```

### 最終到達目標（プログラム修了者に求める力）

- 劇作の**基礎知識**
- 自分のアイデアとニーズを明確に言語化できる**コミュニケーションスキル**
- 国際共同創作に向けた**柔軟な姿勢とオープンマインド**
- 長編戯曲を1本書き上げた**実作経験**

## 各フェーズ詳細

### フェーズ0：導入講座（2026年9月）

**概要**  
劇作の基礎を学ぶ導入コース。座学による知識習得とライティングワークショップの2部構成。ゲストスピーカーを招聘。

**プログラム内容**

| 区分 | 内容 |
|------|------|
| 座学 | 演劇の種類・構造 |
| ゲスト | 前川知大（劇作家）、田中幸子（脚本家）、小説家（TBD） |
| ワークショップ | 他ジャンルのライティング入門、ライティング演習 |

**目標**
- 劇作の基礎知識の習得
- 実際の劇作家のリアルな経験談を聞く（授業では得られないインサイト）
- 座学で得た知識をライティング演習で応用する

---

### フェーズ1：メンターシップ（2026年11月〜2027年4月）

**概要**  
6ヶ月間の1対1メンタリング。月1回のメンターとのセッション。選考通過者10名が長編戯曲を執筆。追加サポートも提供。

**プログラム内容**

| 区分 | 内容 |
|------|------|
| 主活動 | メンターによる1対1指導 |
| 執筆環境 | 「ライターズルーム」（集中執筆スペース）の提供 |
| 資料アクセス | 台本・劇作関連書籍へのアクセス提供 |
| コミュニティ | 同期劇作家との交流活動 |
| 観劇支援 | 公演観劇への財政的支援 |

**目標**
- 執筆習慣の確立
- 長編戯曲1本の完成
- 自作を他者に共有する自信の獲得
- 自分の作品を人前でプレゼンする経験

---

### フェーズ2：プロダクション（2027年9月）

**概要**  
上位5作品を翻訳し、海外劇場でプロフェッショナル制作。国際チームとの共同作業。

**連携パートナー候補**  
**Traverse Theatre**（スコットランド・エディンバラ）  
*A Play, A Pie, and A Pint* および *Class Act* を通じた劇作家支援実績を持つ。そのノウハウと制作チームがこのプロダクション実現に最適。

**プログラム内容**

| 区分 | 内容 |
|------|------|
| 制作形態 | 国際チームとのコラボレーション |
| 成果物 | 新作戯曲のプロフェッショナルプロダクション |

**目標**
- プロとして作品が制作される経験
- 制作チームとのコミュニケーション実践
- 観客の反応を直接受け取る経験
- 日英の制作プロセス・演劇環境・観客反応の比較

## 参加者規模と選考フロー

```
フェーズ0：導入講座
  最大20名参加
      ↓
  選考①（プロット提出ベース）
      ↓
フェーズ1：メンターシップ
  10名選出（2026年11月〜2027年5月）
      ↓
  選考②（執筆ドラフトベース）
      ↓
フェーズ2：プロダクション
  5名選出（2027年9月）
```

**2段階の選考**を通じて5本の長編戯曲を選出。若い劇作家は戯曲を書くだけでなく、プロへのフィードバックの機会と、国際ステージへの扉を得る。

---

## 選考基準

### 選考①（導入講座後）

- 提出したプロットの関連性（Relevance）
- 受講への個人的な動機（Personal Motivation）

### 選考②（メンターシップ後）

- 戯曲の品質（Quality of Play）
- 翻訳可能性（Translatability）
- メッセージ性（Message）
