# 各職能個別要件（一般教養との差分）v1

> 作成日：2026年6月  
> 一般教養プログラムでカバーされる「共通要件」を除いた後の、職能固有の能力・経験・機会の整理  
> 参照元：各職能ToBeモデルv2/v4（SOIL統合版）

---

## 前提：一般教養 vs 職能別の切り分け

```
【一般教養がカバーすること】
  ├── 市場構造の地図（4列マトリックス・転生装置・シナリオA/B）
  ├── ドラマトゥルギー基礎（戯曲の読み方・構造）
  ├── 制作実務基礎（知財・助成金・ハラスメント）
  ├── 日本文化の言語化（英語で語る基礎）
  ├── 英語ピッチ体験（30秒〜2分）
  └── 3職種横断グループワーク

【職能別でカバーすること ← このドキュメントの対象】
  ├── プロデューサー固有の能力・経験・機会
  ├── 演出家固有の能力・経験・機会
  └── 劇作家固有の能力・経験・機会
```

## 1. プロデューサー個別要件

### 1-1. 固有の能力（一般教養で扱わないもの）

| カテゴリ | 要件 | 内容 | シナリオ |
|---------|------|------|---------|
| **スキル** | 戦略的市場選択 | 4列マトリックスを「地図として知る」（一般教養）から「自分のプロジェクトに当てはめて戦略を立てる」へ。どのシナリオ・どの入口・どの壁を越えるかを判断する力 | A・B |
| **スキル** | 転生装置へのアクセス戦略の実践 | NT・Barbican・Public Theatreのキュレーターとどう繋がるか。エジンバラ・アビニョンへの出展をプロデューサーとして設計する。「概念を知る」（一般教養）から「実際にアクセスする」へ | A・B |
| **スキル** | IPライセンス交渉の実践 | IPホルダー（出版社・アニメ会社）との権利交渉。ライセンス条件の読み方・交渉の進め方・弁護士との連携。一般教養では「概念」、職能別では「実践」 | A |
| **スキル** | 国際共同制作のマネジメント | 文化機関との共同制作スケジュール管理・予算設計・スタッフィング。海外と日本の制作システムの違いをマネジメントに落とし込む | A |
| **スキル** | ノンバーバル作品のフォーマット設計 | 再演可能・言語不要・観光向けの作品フォーマットを設計する。STOMPやナンタが示す「コンテンツIP化」の設計思想 | B |
| **スキル** | アーティストマッチング（劇作家×演出家） | 韓国で繰り返し指摘された課題。劇作と演出の深い理解に基づく、戦略的なマッチングの実践 | A |
| **スキル** | 英語ピッチの実践レベル（交渉・プレゼン） | 一般教養では「30秒体験」。職能別では実際の交渉・プレゼン・デック作成・フォローアップまで | A・B |

### 1-2. 固有の経験・機会

| 経験・機会 | 内容 | シナリオ |
|-----------|------|---------|
| 転生装置への実際の訪問・ネットワーキング | NT・Barbican・エジンバラ等への視察。キュレーターとの実際の対話体験 | A・B |
| IPホルダーとの交渉ロールプレイ | 出版社・権利者役との模擬交渉体験（実際の条件・交渉ポイントを使用） | A |
| 実際の国際共同制作への参加・見学 | 現在進行中の日英・日韓・日米共同制作プロジェクトへの同行・観察機会 | A |
| ビジネスプラン作成演習 | シナリオA or Bを想定した実際のプロジェクトプランを作成し、先輩プロデューサー・アドバイザーにプレゼン | A・B |
| エジンバラ・アビニョン参加ツアー（将来） | 転生装置の人たちと出会う「場に出る」体験 | B |

### 1-3. カリキュラム設計のポイント

```
一般教養で「地図を得た」後、プロデューサー職能別では：

【座学フェーズ】
  ├── IPライセンス実務（福井健策弁護士等との深掘り）
  ├── 国際共同制作の設計事例（トトロ・デスノート等の経緯の詳細）
  └── シナリオ別プロデュース戦略の立案

【実践フェーズ】
  ├── ロールプレイ（IPホルダー交渉・転生装置ピッチ）
  ├── 実際のネットワーキング機会（フェスティバル参加・視察）
  └── プロジェクトプランのメンタリング（松田誠・渡辺ミキ等アドバイザーとの面談）
```

## 2. 演出家個別要件

### 2-1. 固有の能力（一般教養で扱わないもの）

| カテゴリ | 要件 | 内容 | シナリオ |
|---------|------|------|---------|
| **知識** | 英国・北米商業演劇のシステムと文法 | ウエストエンド・ブロードウェイの制作システム（リハーサル期間・スタッフ構成・Union・慣習等）の詳細理解。「商業演劇で演出家に何が求められるか」 | A |
| **知識** | フィジカルシアター・ノンバーバルの理論と系譜 | Pina Bausch・DV8・Frantic Assembly・Akram Khan等の作品と思想の深掘り。「なぜ言語なしで感情が成立するか」の理論 | B |
| **スキル** | ノンバーバル演出技術の実践 | 身体・空間・リズム・沈黙・繰り返しのみで意味と感情を作る演出技術。言語に依存しない稽古場のコミュニケーション方法 | B |
| **スキル** | IP制約下での創造性の発揮 | IPの世界観・キャラクター・既存イメージを守りながら、現地観客が自分を投影できる作品に仕上げる演出力。制約を創造性の条件として活かす | A |
| **スキル** | 異言語環境での稽古場マネジメント | 英語・通訳を介した稽古場の進行管理。言葉に頼らない演出指示（感情・イメージ・身体で伝える）。通訳との協働 | A・B |
| **スキル** | 文化的翻訳の実践技術 | 「日本の文化に触れた体験で終わらせない」（木村早智）。普遍的な人間ドラマへの昇華技術を稽古場で実際に実現する | A |
| **スキル** | フリンジ規模での演出設計 | エジンバラ等のフリンジ向け、低予算・短期公演・小空間での演出の最適化。「小さいからこそ光る作品」の設計 | B |

### 2-2. 固有の経験・機会

| 経験・機会 | 内容 | シナリオ |
|-----------|------|---------|
| フィジカルシアター集中ワークショップ | DV8・Frantic Assembly等のメソッドを体験できる国内外のWSへの参加 | B |
| 実際に演出する場の確保（サマースクール内） | 劇作家が書いた短編・devised作品を実際に演出する体験（2〜3日の集中演出実習） | A・B |
| 異言語環境での演出体験 | 英語話者（留学生・在日外国人俳優等）との稽古場体験。通訳を介した演出の実践 | A |
| 優秀な演出家のアシスタント機会 | サマースクールゲスト演出家への質疑・観察。長期的なアシスタント制度への接続 | A・B |
| 作品ビジョンプレゼン（英語） | 自分が演出したい作品のビジョンを英語でキュレーター・プロデューサーに説明する体験 | A・B |

### 2-3. カリキュラム設計のポイント

```
一般教養で「地図を得た」後、演出家職能別では：

【シナリオA向け】
  ├── 商業演劇文法の講義（英国・北米の制作システム解説）
  ├── 戯曲精読・立体化ワーク（テキストから演出プランを作る）
  └── 異言語環境での演出実習

【シナリオB向け】
  ├── フィジカルシアターのメソッド体験WS
  ├── ノンバーバル作品の短期演出実習（俳優なし / 声なしの制約）
  └── フリンジ向け規模感の演出設計演習

【共通】
  ├── 演出家同士のディスカッション（シナリオ選択の対話）
  └── メンター演出家との1on1（アドバイザリーボードを活用）
```

## 3. 劇作家個別要件

### 3-1. 固有の能力（一般教養で扱わないもの）

| カテゴリ | 要件 | 内容 | シナリオ |
|---------|------|------|---------|
| **スキル** | 翻案・アダプテーションの実践技術 | 原作の「魂」を守りながら、現地観客に届く形に変換する技術。原作分析→エッセンス抽出→現代・他文化文脈への移植のプロセス | A |
| **スキル** | 英訳を前提とした戯曲の書き方 | 翻訳されることを意識した文体設計：センテンスの長さ・文化特有の慣用表現の最小化・感情の直接的表現。「翻訳可能な戯曲」を書く訓練 | A |
| **スキル** | 自作をIPとして設計する視点 | 「一回上演して終わり」でなく、英訳→国際リーディング→再演→海外上演のチェーンを見越したドラマトゥルギー設計。続編可能性・キャラクターのIP強度も意識 | A |
| **スキル** | ノンバーバル作品の構造設計（devising） | テキストなしに時間・空間・繰り返し・身体で物語を構造化する技術。演出家・俳優と共同でdeviseするプロセスへの参加 | B |
| **スキル** | 書く技術の集中的な反復練習 | 毎週1本のシーン作成（ソン・ギウン方式）。短い単位からの積み上げによる戯曲技術の習得 | A |
| **スキル** | 国際リーディングへの対応 | 英語話者の俳優に戯曲を読んでもらう体験の準備・対話・フィードバック受け取り | A |

### 3-2. 固有の経験・機会

| 経験・機会 | 内容 | シナリオ |
|-----------|------|---------|
| 翻案ワーク（IPを戯曲化する実習） | 既存の日本の文学・アニメ・民話等を素材に、短編翻案作品を実際に書く体験。演出家・俳優と上演まで進める | A |
| 国際リーディング体験 | 書いた作品（日本語 or 英語）を英語話者の俳優に読んでもらい、反応・フィードバックを受け取る | A |
| 毎日書き続ける習慣の定着（集中期間） | サマースクール期間中、毎日短いシーンを書く。「とにかく書く習慣」の体内化（ドン・マクキャンフィル方式） | A・B |
| devising（共同創作）体験 | 演出家・俳優と共にテキストなし or テキスト最小限の短編作品を共同開発する体験 | B |
| ドラマターグとの対話機会 | 書いた作品についてドラマターグ（ゲスト講師）からフィードバックを受ける1on1セッション | A |
| 劇作家コミュニティの形成 | 同期の劇作家との作品共有・ディスカッション習慣の形成（ARKOモデル：「孤独に書かない」） | A・B |

### 3-3. カリキュラム設計のポイント

```
一般教養で「地図を得た」後、劇作家職能別では：

【シナリオA向け】
  ├── 翻案ワーク集中実習（素材選定→分析→執筆→発表）
  ├── 「英訳を前提とした書き方」のワーク
  └── 国際リーディング体験（英語俳優との対話）

【シナリオB向け】
  ├── devising参加体験（演出家・俳優との共同創作）
  └── ノンバーバル構造設計の入門ワーク

【共通】
  ├── 毎日書く習慣の集中実習（サマースクール全期間）
  ├── 劇作家コミュニティの形成（同期との作品共有）
  └── ドラマターグ・メンターとの1on1
```

## 4. 3職種共通の「機会設計」サマリー

| 機会 | P | D | PW | 一般教養 or 職能別 |
|------|---|---|----|-------------------|
| 転生装置の概念理解 | ◎ | ◎ | ◎ | 一般教養（UNIT 1-B） |
| 転生装置への実際のアクセス | ◎ | ○ | ○ | プロデューサー職能別 |
| 英語ピッチ基礎体験 | ◎ | ◎ | ◎ | 一般教養（UNIT 3-A） |
| 英語ピッチの実践・深化 | ◎ | ◎ | ○ | 各職能別 |
| 日本文化の言語化 | ◎ | ◎ | ◎ | 一般教養（UNIT 2-C） |
| 3職種混合での協働体験 | ◎ | ◎ | ◎ | 一般教養（UNIT 3-B） |
| IP・翻案の実践 | ◎ | ○ | ◎ | 各職能別 |
| ノンバーバル演出・devising | ○ | ◎ | ○ | 演出家/劇作家職能別 |
| メンターとの継続関係 | ◎ | ◎ | ◎ | 各職能別（プログラム全体を通じて） |

> P=プロデューサー / D=演出家 / PW=劇作家
