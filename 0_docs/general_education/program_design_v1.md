## 1. 一般教養プログラムの位置づけ（再定義）

```
一般教養プログラム ＝ 3職能が「同じ地図を見て・同じ言語で話せる」ようになる場
```

- 3職能全員が受講する共通土台
- JPASNの正会員（社会人）も参加可能（学生と社会人が共通言語を得る場）
- 選考①の筆記テスト範囲に連動する
- **職能別プログラムの前**に受講することで、職能別学習の解像度を上げる

### 棲み分け原則（プロデューサー座学との区別）

| 一般教養 | 職能別プログラム |
|---------|---------------|
| WHY（なぜこの市場か・なぜこの人材か） | HOW（どうやって実現するか・どう売るか） |
| 地図（世界のどこにいるか・どこを目指すか） | 技術（具体的な実装方法） |
| 概念レベルの共通知識 | 実践スキルレベルの専門知識 |

## 2. 共通要件の抽出（3職能横断分析）

### 3職能すべてで◎の要件

| 共通要件 | プロデューサー | 演出家 | 劇作家 | 一般教養での扱い |
|---------|-------------|--------|--------|----------------|
| コラボレーション（M-1/M-3） | M-1 ◎ | M-1/M-2 ◎ | M-3 ◎ | グループワーク・職種横断対話の場 |
| 自分の文化への誇りと相対化（M-2/K-4） | M-2/K-2 ◎ | M-4/K-4 ◎ | M-2/K-4 ◎ | 日本文化の言語化ワーク |
| 現場体験（E-1/E-4） | E-1 ◎ | E-4 ◎ | E-1 ◎ | 劇場見学・現場体験（事前プログラム） |
| ドラマトゥルギー基礎（K-1） | K-1 ◎ | K-1 ◎ | K-1 ◎ | 戯曲読解・批評リテラシー入門 |
| 制作実務基礎（K-3/K-5） | K-3 ◎ | K-8 ○ | K-5 ◎ | 知財・助成金・契約の概論 |
| 英語コミュニケーション（S-2/S-5/S-7） | S-2 ◎ | S-5 ◎ | S-7 ○ | 英語ピッチ・自己表現の基礎ワーク |
| 自作・作品を語る力（S-3/S-6） | S-3 ◎ | S-3 ◎ | S-2 ◎ | 言語化ワークショップ |

### A/Bシナリオ統合で追加される共通要件

| 追加共通要件 | なぜ3職種すべてに必要か |
|-------------|----------------------|
| **SOIL市場構造リテラシー** | 「自分はどこを目指しているか」の地図がないと職能別学習の動機が弱い。4列マトリックス・転生装置・3つの入口・シナリオA/Bを3職種全員が知る必要がある |
| **成功転生事例のケーススタディ** | トトロ（A型）・Six/STOMP（B型）・チェルフィッチュ（D型）の転生ルートを3職種視点で読む。「職能ごとにどの場面で何が必要だったか」を分解する |
| **国際フェスティバルの地図** | エジンバラ・アビニョン・各国フリンジの構造とキュレーターの役割を知ることが、シナリオBの現実的な出口理解につながる |
| **英語での日本文化プレゼンテーション** | 3職種すべてで「日本の文化を英語で説明できる」が◎。一般教養で共通ワークとして実施することで、職能別でも応用できる基盤になる |

## 3. 一般教養プログラムの全体構成（案）

### 全体コンセプト：「地図を持ち、言語を得る」

```
UNIT 1：地図を持つ（WHY・WHERE）
  ├── 1-A. 日本の舞台芸術が置かれている位置
  └── 1-B. 世界市場の構造と転生装置

UNIT 2：言語を得る（WHAT・HOW TO SAY）
  ├── 2-A. 演劇の共通言語（ドラマトゥルギー基礎）
  ├── 2-B. 制作の共通言語（知財・法務・助成金）
  └── 2-C. 自分の文化を語る言語（日本文化の言語化）

UNIT 3：動かしてみる（TO WHOM・HOW）
  ├── 3-A. 英語での作品ピッチ体験
  ├── 3-B. 3職種横断グループワーク
  └── 3-C. 転生事例のケーススタディ（ロールモデルとの接続）
```

## 4. 各ユニットの内容設計

### UNIT 1-A：日本の舞台芸術が置かれている位置

| 項目 | 内容 |
|------|------|
| **テーマ** | 「なぜ今、海外展開なのか」——業界の現状と課題 |
| **カバーする内容** | ①日本の演劇業界の現状（市場規模・海外展開の少なさ）　②SEEDが生まれた背景（業界課題・SOILとの連携）　③他国との比較（韓国・英国・米国の育成環境） |
| **ゴール** | 参加者が「自分がこの事業に関わる意味」を言葉にできる |
| **形式** | レクチャー（60〜90分）+ ディスカッション |
| **担当想定** | 野村善文（SEED事務局長）、横堀応彦（リサーチ統括） |

---

### UNIT 1-B：世界市場の構造と転生装置

| 項目 | 内容 |
|------|------|
| **テーマ** | 「世界の舞台芸術市場はどう動いているか」——SOIL戦略フレームワーク入門 |
| **カバーする内容** | ①4列マトリックス（国際IP/商業招聘/現代演劇/文化資産）　②転生装置の概念と主要機関（NT・Barbican・Public Theater等）　③3つの入口（A場所貸し/B共同制作/Cフリンジ）　④シナリオA/Bの概要と職能ごとの役割 |
| **ゴール** | 参加者が「自分がどのシナリオで働きたいか」を選択・説明できる |
| **形式** | レクチャー（90分）+ グループディスカッション（職種混合） |
| **担当想定** | 横堀応彦 ＋ SOILリサーチャー（時田曜子・小川絵梨子等） |

---

### UNIT 2-A：演劇の共通言語（ドラマトゥルギー基礎）

| 項目 | 内容 |
|------|------|
| **テーマ** | 「戯曲を読む・語る・分析する」——3職種の共通基盤 |
| **カバーする内容** | ①ドラマトゥルギーの基礎概念（コンフリクト・アーク・サブテキスト）　②戯曲の読み方（劇作家・演出家・プロデューサーそれぞれの読み方の違い）　③批評リテラシー入門（どう批評するか・批評をどう活かすか） |
| **ゴール** | 「この戯曲の何が世界に届くか/届かないか」を3職種が共通言語で議論できる |
| **形式** | 戯曲リーディング + 分析ワークショップ（2〜3時間） |
| **担当想定** | 倉沢（劇作家担当）、川名（演出家担当） |

---

### UNIT 2-B：制作の共通言語（知財・法務・助成金）

| 項目 | 内容 |
|------|------|
| **テーマ** | 「演劇の仕事に必要な社会人的知識」——3職種が最低限知るべき実務 |
| **カバーする内容** | ①著作権・知的財産の基礎（戯曲・楽曲・振付の権利）　②海外展開における契約・ライセンスの概要　③助成金・資金調達の種類と申請の考え方　④ハラスメント対策・労働倫理（JPASNとの連携） |
| **ゴール** | 「プロデューサーに任せる」から「自分でも最低限理解して判断できる」へ |
| **形式** | レクチャー（ゲスト講師中心）+ Q&A |
| **担当想定** | 福井健策弁護士（知財・著作権）、遠藤真有美（JPASN連携・会計） |

---

### UNIT 2-C：自分の文化を語る言語（日本文化の言語化）

| 項目 | 内容 |
|------|------|
| **テーマ** | 「自分の文化を英語で・第三者に・説得力を持って語る」——海外展開の土台 |
| **カバーする内容** | ①日本演劇の歴史概要（能・歌舞伎・新劇・現代演劇の系譜）　②日本の「精神的価値観」の言語化（アニミズム・間・もののあわれ等）　③「なぜ日本の演劇が世界に届くのか」を自分の言葉で説明する練習 |
| **ゴール** | 海外のプロデューサー・キュレーターに「日本の演劇の何が面白いか」を5分で説明できる |
| **形式** | レクチャー + 個人発表ワーク（2〜3分スピーチ）|
| **担当想定** | 横堀応彦（リサーチ統括）、SEEDリサーチャーOB（小栗・阿部等） |

---

### UNIT 3-A：英語での作品ピッチ体験

| 項目 | 内容 |
|------|------|
| **テーマ** | 「30秒で作品・プロジェクトを売る」——全職種が持つべき最低限の英語発信力 |
| **カバーする内容** | ①作品・プロジェクトを1センテンスで説明する（logline）　②なぜ今・なぜ世界にを英語で語る　③フィードバックを受け取り改善する |
| **ゴール** | 全員が英語で30秒〜2分のピッチを実際にやってみる体験 |
| **形式** | ワークショップ（反復練習中心）|
| **担当想定** | 赤羽（プロデューサー担当）、英語ファシリテーター |

---

### UNIT 3-B：3職種横断グループワーク

| 項目 | 内容 |
|------|------|
| **テーマ** | 「3職種が一緒に考える——どの作品を・どこに・どうやって届けるか」 |
| **カバーする内容** | プロデューサー・演出家・劇作家の混合チームで仮想プロジェクト（例：架空の日本発作品をシナリオA or Bでどう展開するか）を設計する |
| **ゴール** | 3職種が「お互いの役割が何か」「協働の場面でどんな対話が必要か」を体感する |
| **形式** | グループワーク（半日〜1日）+ 発表 |
| **担当想定** | 赤羽・野村（全体統合）、各職能担当者がファシリテーション支援 |

---

### UNIT 3-C：転生事例ケーススタディ

| 項目 | 内容 |
|------|------|
| **テーマ** | 「実際にどう転生したか」——ロールモデルの転生ルートを解剖する |
| **カバーする内容** | ①トトロ/NT共同制作の経緯（シナリオA事例）　②STOMP/ナンタの長期展開（シナリオB事例）　③チェルフィッチュ/野田地図の国際展開（参考事例）　④各事例で3職種それぞれが何をしたか |
| **ゴール** | 「自分がこの事例のプロデューサー/演出家/劇作家だったら何をしたか」を語れる |
| **形式** | ケーススタディ + ディスカッション |
| **担当想定** | 横堀応彦 ＋ 海外リサーチャー報告との接続 |

## 5. プログラム全体のスケジュールイメージ

```
【事前プログラム（オンライン）】
  ├── UNIT 1-A：日本の舞台芸術の現状（事前視聴 + 課題）
  └── UNIT 2-B：制作の共通言語（事前視聴）

【サマースクール本体（対面）】
  Day 1：
    ├── UNIT 1-B：世界市場の構造と転生装置（午前）
    └── UNIT 2-A：ドラマトゥルギー基礎（午後）
  
  Day 2：
    ├── UNIT 2-C：自分の文化を語る言語（午前）
    └── UNIT 3-A：英語ピッチ体験（午後）
  
  Day 3〜：各職能別プログラムへ（職能別で深化）
  
  最終Day：
    ├── UNIT 3-B：3職種横断グループワーク
    └── UNIT 3-C：転生事例ケーススタディ
```

## 6. 選考①筆記テストとの連動

一般教養プログラムの内容が選考1の筆記テスト範囲となる。  
テスト設計の方向性：

| ユニット | テスト観点 |
|---------|-----------|
| UNIT 1-A | 業界課題の認識・SEEDが存在する意味の理解 |
| UNIT 1-B | 4列マトリックス・転生装置・シナリオA/Bの基本理解 |
| UNIT 2-A | 戯曲の基礎的な読解・ドラマトゥルギー概念の理解 |
| UNIT 2-B | 著作権・助成金・ハラスメント対策の基礎知識 |
| UNIT 2-C | 日本文化・演劇史の基礎知識 |

→ テスト詳細設計は `selection` エージェントに引き継ぐ

## 7. プロデューサー座学との棲み分け（重要）

| テーマ | 一般教養 | プロデューサー座学 |
|--------|---------|-----------------|
| 知財・著作権 | 概念・全員に必要な基礎（WHY/WHAT） | IPライセンス交渉の実践（HOW） |
| 世界市場の構造 | 4列マトリックス・転生装置の概念 | 具体的なアプローチ戦略・ネットワーク構築 |
| 英語コミュニケーション | 30秒ピッチの体験 | 実際の交渉・プレゼン・契約文書 |
| 助成金・資金調達 | 種類と考え方の概論 | 申請書の書き方・戦略的資金設計 |
| ケーススタディ | 転生事例を3職種視点で読む | 自分のプロジェクトへの応用・戦略設計 |
