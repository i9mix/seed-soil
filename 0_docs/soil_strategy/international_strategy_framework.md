## SEED と SOIL の位置づけ

| 事業 | 役割 |
|------|------|
| **SEED** | 海外展開できる人材（プロデューサー・劇作家・演出家）を育成する |
| **SOIL** | SEEDが育てた人材・作品が実際に海外展開するための市場・回路を耕す |

- **SOIL → SEED**：SOILが海外ピッチ等で得た市場ニーズをSEEDのカリキュラムに反映
- **SEED → SOIL**：SEEDで育成した人材・作品をSOILの海外派遣プログラムへ送り込む

このフレームワークは両事業の根幹となる市場認識・構造認識を定義する。

## 1. 4列の市場構造

```
国際IP → 商業招聘 → 現代演劇 → 文化資産
```

| 列 | 定義 | 事例 |
|----|------|------|
| **国際IP** | 複製・転用できる資産 | ライオンキング、トトロ |
| **商業招聘** | 資本で動く市場 | STOMP、ナンタ、オペラ座の怪人 |
| **現代演劇** | 商品化される前・文化資産になる前。ボリュームが最大。**勝負のステージング領域** | チェルフィッチュ、野田地図 |
| **文化資産** | 時間が証明した希少性 | 能・歌舞伎、ピナ・バウシュ、鈴木メソッド |

---

## 2. マトリックスの軸定義

### 横軸：複製芸術 ↔ 希少芸術
- **左（複製芸術）**：ターゲットの幅が広い。ライセンス・フランチャイズ・IP展開ができる
- **右（希少芸術）**：深度が深い。特定のアーティスト・身体・文脈に価値が宿る

### 縦軸：訴求幅（上から下へ）
1. **ファミリー含む全年齢**（子供〜大人・世界中どこでも）
2. **観光客含む幅広い対象**（旅行者・都市エンタメ層）
3. **特定の国・文化に帰属する人たち**（文化的文脈を共有するコミュニティ）
4. **演劇人**（演劇界・批評家・フェスティバル層）

---

## 3. VOID / DEAD ZONEの定義

### VOID（構造的空白・市場が成立しない領域）
- **右上VOID**：文化資産×ファミリー全年齢 → わざわざ海外から招聘する理由がない
- **左下VOID**：商業招聘×演劇人 → 商業展開しようとしているのに演劇人にしか届かない

### DEAD ZONE
- 現代演劇の中で**商業でも文化でも振り切れていない中途半端な領域**
- 自主開催フリンジ枠。誰に届けたいかが曖昧
- 左側（商業）にも右側（文化資産）にも存在する

---

## 4. 越えるべき壁

### 左側（商業ゾーン）の壁
- **資本の壁**：Cirque・ライオンキング級の投資ができるか
- **対象年齢の壁**：ファミリーから観光客へ
- **作品知名度の壁**：オペラ座・レ・ミゼラブル級の認知があるか
- **言語・通例理解の壁**：ノンバーバル系がここで止まる
- **多言語・多文化対応の壁**
- **歴史観の壁**：ハミルトン・Come from Awayはここで止まる

### 右側（現代演劇・文化ゾーン）の壁
- **人脈の壁**：転生装置の人たちに出会えるか ← **最も高い壁**
- **マーケティングの壁**
- **ランニングコストの壁**
- **コストの壁**
- **時間の壁**

## 5. 転生装置（最重要概念）

### 定義
文化機関と商業劇場の中間に位置する劇場・機関。**広い顧客基盤を持つ文化機関が商業転生の入口になる**。

> 商業にとっての掘り出し物、国際フェスティバルキュレーションにとってのカタログを果たす。

### 主要な転生装置

| 都市 | 機関 |
|------|------|
| **ロンドン** | National Theatre、Barbican、Arts Theatre、Donmar Warehouse、Young Vic |
| **ニューヨーク** | Public Theater、Second Stage、Ford's Theatre |

### 一番高い壁
**「転生装置の人たちに出会えるかどうか」**  
資本でも言語でも解決できない。エディンバラ・アビニョンに出る理由はそこにいる「転生装置の人たち」に会うため。

## 6. 転生ルートの事例

| パターン | ルート | 特徴 |
|----------|--------|------|
| **トトロパターン（機関型転生）** | NT → バービカン → ウエストエンド商業 | 既存IPで文化機関と共同制作。2〜3年で転生 |
| **Sixパターン（フリンジ型転生）** | エディンバラ・フリンジ → アーツシアター → ウエストエンド → ブロードウェイ | 予算ほぼゼロからスタート |
| **War Horseパターン（NT制作型）** | NT制作 → ブロードウェイ商業成功 → ライセンス展開 | 文化機関が商業転生した典型 |
| **Hamiltonパターン（Public Theater型）** | Public Theater → ブロードウェイ | 英語圏以外への展開はほぼゼロ。歴史観の壁未突破 |
| **MHEパターン（資本・外交型）** | ソウル → 直接ブロードウェイ | 転生装置を経由しない稀なケース。国家戦略・資本力で直接転生 |
| **千と千尋・野田地図パターン（場所貸し型）** | 文化機関に場所代を払って自主公演 | **お金で解決できる唯一の入口** |

## 8. 4つの戦略シナリオ

### シナリオA：トトロパターン
**既存の有名IPを利用したファミリー向けミュージカルを海外機関と共同開発し市場展開**

- STEP 1：既存IPを利用した北米・英国の文化施設との共同制作での作品開発を実施
- STEP 2：IPの集客力と資本力を使って、ファミリー向け作品としてマーケットに接続

**必要な人材：**
- プロデューサー：海外文化機関とのネットワークを持つ国際共同制作の経験者
- 演出家：英国・北米の商業演劇文法を理解している人
- 劇作家：IPの世界観を守りながら現地観客に届ける翻案力

---

### シナリオB：ストンプパターン
**日本国内インバウンド観光をテストマーケットにし、海外の観光需要を狙う**

- STEP 1：国内インバウンド対象に観光消費に寄せた作品を開発
- STEP 2：フリンジフェスティバルに出展・作品を見てもらう
- STEP 3：ノンバーバル作品として世界で観光客向けに行われる作品としてIP展開

**必要な人材：**
- プロデューサー：フリンジ・フェスティバルのネットワークを持つ人
- 演出家：ノンバーバル・身体表現のスペシャリスト
- 劇作家：（言語不要なのでほぼ不要）

---

### シナリオC：木下歌舞伎パターン（長期投資）
**古典・伝統芸能を現代的に再解釈して海外へ**

時間がかかる（10〜20年単位）が、**他の国が絶対に複製できない差別化領域**。  
能・歌舞伎・文楽・舞踏は数百年かけて積み上げた文化資材。韓国もアメリカもヨーロッパも持っていない。

**必要な人材：**
- プロデューサー：文化外交・公的支援のネットワークを持つ人
- 演出家：古典の文法を理解しつつ現代的解釈ができる人
- 劇作家：伝統様式を現代語に翻訳できる人

---

### シナリオD：チェルフィッチュパターン
**国内で評価された作品を国際的文脈に載せてフェスティバル招聘→羽ばたく**

**必要な人材：**
- プロデューサー：国際フェスティバルのキュレーターと繋がれる人
- 演出家：社会的文脈・時代性を持った作品を作れる人
- 劇作家：日本社会の今を普遍的言語で語れる人

## 10. 足りない人・構造

### 人材層
- 国際共同制作の経験があるプロデューサーがほぼいない
- 海外文化機関のキュレーターと個人的な信頼関係を持つ人がいない
- 英語で交渉・契約できる演劇専門の弁護士・エージェントがいない
- IPホルダー（出版社・アニメ会社）と演劇界を繋ぐブリッジ人材がいない

### 構造層
- 転生装置（NT・バービカン等）との継続的な関係構築の仕組みがない
- 国際共同制作のための資金スキームがない（文化庁の補助金は国内向け）
- 作品を海外に持っていく際のツアー資金が確保できない
- IPライセンスの国際交渉窓口が分散している

### 情報・データ層
- 「どの市場を狙うべきか」の分析が業界に存在しない
- 海外の転生装置がどんな作品を求めているかのインテリジェンスがない
- 成功・失敗事例の共有がされていない

### 最も本質的な問題
**「誰がこれをドライブするか」が不明確**  
文化庁・国際交流基金・民間プロデューサー・IPホルダーがそれぞれバラバラに動いていて、統合的な戦略を持つ主体がいない。

## 11. 重要概念の定義

| 概念 | 定義 |
|------|------|
| **転生装置** | 文化機関と商業劇場の中間に位置し、作品を商業転生させる劇場・機関 |
| **DEAD ZONE** | 誰に届けたいかが曖昧な自主開催フリンジ枠 |
| **VOID** | 構造的に市場が成立しない領域 |
| **勝負のステージング領域** | 現代演劇の上段2行。世界中のプロデューサー・キュレーターが同時に見ている唯一の場所 |
| **場所貸し自主公演** | 文化機関に場所代を払って自主公演するモデル。お金で解決できる唯一の入口 |
| **訴求幅** | 縦軸の本質。ファミリー全年齢から演劇人まで、誰に届くかの幅 |

## カリキュラム設計への示唆

### 各シナリオが示す「育てるべき人材」の要件

| シナリオ | プロデューサー | 演出家 | 劇作家 |
|----------|---------------|--------|--------|
| A（トトロ型） | 海外文化機関ネットワーク・国際共同制作経験 | 英国・北米商業演劇文法の理解 | IPの翻案力・現地観客への届け方 |
| B（ストンプ型） | フリンジ・フェスティバルネットワーク | ノンバーバル・身体表現 | （ほぼ不要） |
| C（歌舞伎型） | 文化外交・公的支援ネットワーク | 古典文法×現代的解釈 | 伝統様式の現代語翻訳 |
| D（チェルフィッチュ型） | 国際フェスティバルキュレーターとの繋がり | 社会的文脈・時代性を持つ作品制作 | 日本社会の今を普遍的言語で語る力 |

### 共通して必要な素養
- 英語による交渉・プレゼン能力
- 国際共同制作のプロセス理解
- IPライセンス・著作権の基礎知識
- 転生装置（文化機関）のキュレーターとの関係構築
